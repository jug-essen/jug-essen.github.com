Die JUG Essen ist eine von weltweit vielen Java User Groups. Sie organisiert Veranstaltungen im Bereich Java wie etwa Vorträge und Stammtische. Dabei geht es nicht allein um die Sprache Java, sondern um Softwareentwicklung bei der Java oder die JVM einen prominenten Platz einnimmt. Eingeladen sind alle Java-Interessierten, egal ob Profis oder Privatleute. Es gibt keine Mitgliedschaft; die Teilnahme an den Veranstaltungen ist kostenlos. Wir freuen uns natürlich über jede Hilfe von Organisationen, die die JUG fördern wollen
